# Cloud Router

## What is BGP?
 - Border Gateway Protocol (BGP) is used for network discovery of IP subnets.
 - It simplifies network management by automatically creating routes when new subnets are discovered.

## What is ASN?
Autonomous System Number (ASN) is an unique identifier used to register and recognize routers.

## Scope
 - Regional will only discover routes in vpc having a presence in the same region
 - Global will discover routes in any regions of the GCP infrastructure
